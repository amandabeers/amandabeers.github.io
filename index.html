About Me

In my previous role as a Structural Engineer I always enjoyed working on automating a process or design calculation in Excel. I decided to learn more about coding and starting doing tutorials in my free time. After getting several tutorials deep in JavaScript I realized how much I loved pitting my mind against coding problems.

I decided to enroll in GA's Software Engineering Immersive after speaking with several friends who are GA alumni. I've always enjoyed learning and the constant stream of new information at GA was energizing. I'm excited to put my new skills along with my detail-oriented and curious nature to use working with a team of committed developers.

Bucket List

A full stack web application for creating and editing a bucket list. Bucket List was built while working on a team of three developers and features include crossing off items on the list as you complete them and filtering by completed items. This app was building using MongoDb, Mongoose, Express, JavaScript, jQuery, and Handlebars.

Game Shelf

A full stack web application built using SQL, Ruby on Rails, JavaScript, jQuery, and Handlebars. This app allows a user to add board games to their shelf from a global games resource. Features include filtering your games based on number of players and adding/updating notes on your games from your own experiences playing them.

Tic Tac Toe

A tic tac toe game built using JavaScript, jQuery, HTML 5, and Bootstrap. The app allows you to sign in and play tic tac toe with a friend or against yourself. Past game states are stored and can be loaded by the id. If the game wasn't finished, you can play it through!
