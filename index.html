Skills

Gloomhaven Character Log

The Gloomhaven Character Log is a companion application for the game Gloomhaven which is a role playing game meant to be played over many sessions. This app allows a user to create characters from the starter classes in Gloomhaven and keep track of character information and stats from game to game. This full-stack web application was built using React.js, React Router, JSX, Bootstrap, Ruby on Rails, and PostgreSQL.

Bucket List

Bucket List is a full stack web application for creating and editing a bucket list. It was created while working on a team of three developers and features include crossing off items on the list as you complete them and filtering by completed items. This app was building using JavaScript, jQuery, Handlebars, HTML5, Bootstrap, MongoDb, Mongoose, and Express.

Game Shelf

Game Shelf is a full stack web application built using JavaScript, jQuery, Handlebars, HTML5, Bootstrap, Ruby on Rails, and PostgreSQL. This app allows a user to add board games to their shelf from a global games resource. Features include filtering your games based on number of players and adding/updating notes on your games from your own experiences playing them.

Tic Tac Toe

A tic tac toe game built using JavaScript, jQuery, HTML5, and Bootstrap. The app allows you to sign in and play tic tac toe with a friend or against yourself. Past game states are stored and can be loaded by the id. If the game wasn't finished, you can play it through!
